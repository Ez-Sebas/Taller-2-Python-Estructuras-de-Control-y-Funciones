# 📘 Taller 2: Python — Estructuras de Control y Funciones


## 📌 Descripción del proyecto

Este repositorio contiene una colección de ejercicios desarrollados en Python como parte del proceso de aprendizaje en programación durante el primer trimestre de la formación en Análisis y Desarrollo de Software.

El trabajo está organizado en diferentes secciones que reúnen ejercicios enfocados en el desarrollo de la lógica de programación y el uso de las principales estructuras del lenguaje Python.

Cada ejercicio fue diseñado para aplicar conceptos fundamentales de programación y fortalecer la capacidad de resolver problemas mediante algoritmos.

Como parte final del taller se desarrolló un programa más completo que implementa un sistema básico de gestión de biblioteca.

---

## 🎯 Objetivo

Fortalecer la lógica de programación mediante la implementación de ejercicios progresivos organizados por secciones temáticas.

---

## 🗂️ Organización del taller

El taller está dividido en diferentes secciones que agrupan ejercicios relacionados con los temas vistos durante el proceso de aprendizaje.

Las primeras secciones contienen ejercicios enfocados en la práctica de los fundamentos del lenguaje Python, mientras que las secciones posteriores incorporan estructuras más complejas y una mejor organización del código.

Cada archivo dentro de las secciones corresponde a un ejercicio independiente que implementa una solución a un problema específico de programación. Cada ejercicio o algortimo se ejecuta manualmente uno por uno.

---

## 📚 Sistema de gestión de biblioteca

Como parte final del trabajo se desarrolló un programa más completo llamado **Sistema de Gestión de Biblioteca**.

Este programa permite administrar un catálogo de libros utilizando estructuras de datos en Python.

Para almacenar la información se utiliza una **lista de diccionarios**, donde cada libro contiene los siguientes datos:

* identificador único (ID)
* título del libro
* autor
* año de publicación
* estado de disponibilidad

El sistema permite realizar diferentes operaciones sobre el catálogo de libros mediante funciones programadas en Python.

Entre las funcionalidades implementadas se encuentran:

* registro de nuevos libros
* visualización del catálogo de libros
* búsqueda de libros por título o autor
* préstamo de libros
* devolución de libros
* eliminación de registros del catálogo
* consulta de libros por autor
* generación de estadísticas del sistema
* exportación del catálogo a un archivo de texto

Este programa integra varios de los conceptos trabajados durante los ejercicios del taller, especialmente el uso de funciones, listas, diccionarios y estructuras de control.
